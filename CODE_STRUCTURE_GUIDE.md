Prompt for Claude

You are a senior front-end engineer and UI/UX designer.

Build a complete To-Do List web application following the requirements below.

Project Structure

The project must contain exactly 3 files:

index.html
style.css
script.js

Do not create any additional files, folders, libraries, frameworks, build tools, or dependencies.

Design Requirements

Create a minimalistic and modern design.

Requirements:

Clean and professional appearance
Plenty of whitespace
Simple color palette
Easy-to-read typography
Mobile responsive
Modern card-based layout
Subtle shadows and rounded corners
No clutter or unnecessary visual elements
Focus on usability and simplicity

The interface should feel similar to modern productivity applications.

UI Requirements

The UI must be intuitive and easy to understand.

Include:

Header
Application title
Short subtitle
Task Input Section
Text input field
Add Task button
Task List Section

Each task should display:

Task text
Complete button
Delete button
Task States

Incomplete tasks:

Normal appearance

Completed tasks:

Strikethrough text
Reduced opacity
Visual indication of completion
Functionality Requirements

Implement all functionality using vanilla JavaScript.

Add Task
User can type a task
Clicking Add Task creates a new task
Pressing Enter also adds a task
Empty tasks are not allowed
Complete Task
User can mark a task as completed
User can toggle completion on and off
Delete Task
User can remove a task
Local Storage
Save tasks automatically
Restore tasks when page reloads
Persist completion status
Code Quality Requirements
HTML
Semantic HTML5 structure
Proper accessibility attributes
Well-organized markup
CSS
Clean and maintainable
Use CSS variables for colors
Responsive layout
Consistent spacing system
JavaScript
Modular and readable
Use functions where appropriate
Avoid duplicated code
Include comments for important logic
Responsive Design

The application must work properly on:

Desktop
Tablet
Mobile devices

Ensure:

Buttons remain accessible
Text remains readable
Layout adapts smoothly
Output Format

Provide the code in this exact order:

index.html
...
style.css
...
script.js
...

Do not provide explanations. Output only the complete code for all three files.
